# travel-system
基于Vue3的旅游管理与推荐系统【2026原创】

## 一、介绍
本项目是一个基于Flask+Vue3框架开发的旅游景点管理与推荐系统，旨在为用户提供便捷的旅游景点查询、浏览、收藏、点赞、评论以及个性化推荐服务。系统采用前后端分离架构，后端使用Flask轻量级Web框架提供RESTful API接口，前端采用Vue3框架构建响应式用户界面。系统实现了用户注册登录、景点信息管理、收藏点赞评论、协同过滤推荐算法等核心功能。管理员可以管理用户信息和景点数据，普通用户可以浏览景点详情、参与互动，并通过"猜你喜欢"功能获得个性化推荐。系统支持按省份、景点名称进行模糊查询，浏览量实时统计，图片上传等功能，为用户提供全面的旅游信息服务。
